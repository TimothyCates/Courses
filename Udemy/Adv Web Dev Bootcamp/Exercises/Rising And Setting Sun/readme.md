# Rising and Setting Sun

## Goal

To produce a simple animation of a yellow circle, the sun, making an arch in a sky-box that's colors change as this animation occurs.

## Starter Code

### Html

```html
<div class="sun"></div>
<div class="grass"></div>
```

### Css

```css
body {
  margin: 0 0 0 0;
  background-color: #2c3e50;
}

.grass {
  background: green;
  height: 600px;
  width: 100%;
  position: absolute;
}

.sun {
  background: #f1c40f;
  width: 200px;
  height: 200px;
  border-radius: 50%;
}
```
